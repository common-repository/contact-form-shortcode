Contact Form Shortcode is a simple yet easy to use WordPress contact form.

== Description ==

Contact Form Shortcode is a simple contact form for WordPress. 
Just put <code>[contact]</code> into your post and fill out the settings under Settings > Contact Form SC.

Here are some fetures that Contact Form Shortcode includes.
<ol>
	<li>Custom e-mail</li>
	<li>Custom Anti-Spam Question and Answer</li>
	<li>Custom message after the e-mail is sent</li>
</ol>

And more...

Please support the plugin author by checking the "Support Author" button in the settings.

== Installation ==

1. Go to the Plugins> Add New> Upload in you admin dashboard.
2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.

==  ==

= Settings =
The settings page is under settings in the admin dashboard.

= Shortcode =
The shortcode is <code>[contact]</code>, and can be put in posts, pages, and widgets.

= Error Message =
Their is no way to edit the error messages in the settings. They are all default.

= Field Names =
Their is no way to edit the field names in the settings. They are all default.
